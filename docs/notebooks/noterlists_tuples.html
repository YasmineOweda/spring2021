.sort()¶

.sort() er en metode i list objektet.
Generelt skal i undgå at bruge den, og istedet bruge sorted().
Grund: * sorted() kan bruges på alle ‘iterable objects’, .sort() kan kun bruges på lists.
* sorted() ændre ikke i den oprindelige liste. .sort() gør.

Eksempel med string

List comprehensions¶

List Comprehensions provide an elegant way to create new lists.
The following shows the basic structure of a list comprehension:
[110]:

l = [x for x in [1, 2, 3, 4]]

[111]:

l

[111]:

[1, 2, 3, 4]

Et for loop der skrives som dette:

[112]:

l = []
for i in range(1, 10):
    l.append(i)
l

[112]:

[1, 2, 3, 4, 5, 6, 7, 8, 9]

Kunne skrives mere elegangt med en list comprehension

[113]:

[i for i in range(1, 10)]

[113]:

[1, 2, 3, 4, 5, 6, 7, 8, 9]
